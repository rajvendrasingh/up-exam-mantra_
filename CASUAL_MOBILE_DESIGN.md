## 🚀 Animations

### Slide Up:
```css
@keyframes slideUp {
  from { transform: translateY(100%); }
  to { transform: translateY(0); }
}
```

### Pulse:
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
```

### Press:
```css
button:active {
  transform: scale(0.98);
}
```
