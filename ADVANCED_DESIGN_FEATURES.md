### 🎨 Design System

#### **Color Palette**
```css
Primary: #6366f1 (Indigo)
Secondary: #8b5cf6 (Purple)
Accent: #ec4899 (Pink)
Success: #10b981 (Green)
Warning: #f59e0b (Amber)
Danger: #ef4444 (Red)
```

#### **Gradients**
- Primary: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- Success: `linear-gradient(135deg, #10b981 0%, #059669 100%)`
- Warning: `linear-gradient(135deg, #f59e0b 0%, #d97706 100%)`
- Danger: `linear-gradient(135deg, #ef4444 0%, #dc2626 100%)`

#### **Shadows**
- Small: `0 10px 30px rgba(0, 0, 0, 0.1)`
- Medium: `0 20px 40px rgba(0, 0, 0, 0.15)`
- Large: `0 25px 50px rgba(0, 0, 0, 0.2)`
- Colored: `0 8px 20px rgba(102, 126, 234, 0.3)`

#### **Border Radius**
- Small: 12px
- Medium: 15px
- Large: 20px
- XL: 25px
- Pills: 50px

#### **Animations**
```css
fadeIn - Smooth fade in
fadeInUp - Slide up with fade (0.6s with stagger)
scaleIn - Scale animation
float - Floating background shapes (6s-8s)
shimmer - Loading effect
pulse - Pulsing animation
bounce - Bouncing effect
```

### 🎯 Advanced Features

#### **1. Animated Background**


#### **3. Hover Effects**
- Transform: translateY(-8px to -10px)
- Scale: 1.02
- Shadow increase

- Smooth transitions (0.3s ease)

#### **4. Typography**
- Font: Inter (system fallback)
- Weights: 400, 500, 600, 700, 800, 900
- Sizes: 0.85rem to 4rem
